### 1. 프로젝트명 : 영진전문대학 카풀 시스템

--------

### 2, 주       제 : 영진전문대학 재학생들을 위한 카풀 애플리케이션 및 웹 사이트 제작

------------------

### 3. 주제선정배경 
## (1) 카풀 앱을 이용한, 원거리 통학하는 재학생들의 교통비 부담 해소
## (2) 카풀 앱을 이용한, 자가용 통학하는 재학생들의 기름값 부담 해소
## (3) 카풀 앱을 이용한, 서로 다른 계열 재학생들 간 친밀도 향상 유도

------------------------

### 4. 벤치마킹
## (1) 벤치마킹 선정기준   
# -스토어 기준 다운로드 수 1천 이상   
# -사용 범위가 대학생을 위한 캠퍼스용 카풀   
# -기능 선정에 참고가 될만한 앱

## (2) 벤치마킹 대상   
# - POOLUS : 선정기준 1,3에 해당   
# - LUXI : 선정기준 1,3에 해당   
# - 이 근처 합승 : 선정기준 1,3에 해당   
# - K'POOL : 선정기준 2,3에 해당   
# - 캠퍼스카풀 : 선정기준 2,3에 해당   
# - 직풀 : 선정기준 1,2,3에 해당 

## (3) 벤치마킹 결과   
# - 자료저장소 참고

------------------------
### 5. 개발 도구
## (1) 안드로이드 스튜디오 : 앱 구축
## (2) JSP : 웹 구축
## (3) MYSQL : 데이터베이스 구축

---------------------------

### 6. API사용
## -T-MAP API 사용

-------------------------------

### 7. 예상 결과물
## (1) 사용자 정의 : 영진전문대학 재학생
## (2) 사용자 분류 : 차량 미소지자,차량 소지자, 택시 탑승자
## (3) 드라이버 경로에 카풀 탑승을 원하는 사용자가 참여 신청
## (4) 택시 방을 개설하여 택시에 탑승을 원하는 사용자들 참여
## (5) 앱은 카풀에 관련된 핵심적인 기능관련 기능들 만을 집약
## (6) 웹은 카풀에 관련된 사용자와의 커뮤니케이션기능을 집약

-------------------------

### 8. 예상 기대효과
## (1) 직접    
# -재학생들의 교통비 및 유류비 절감 효과    
# -서로 다른 계열 간의 친밀도 향상 효과    
# -재학생들의 시간낭비없는 빠른 등하교 가능
## (2) 간접    
# -팀원들의 안드로이드 프로그래밍에 대한 이해도 증가    
# -팀원들의 JSP 웹 프로그래밍에 대한 이해도 증가    
# -팀원들의 데이터베이스 구성에 대한 이해도 증가    
# -네트워크 서버/클라이언트 구조에 대한 이해도 증가    
# -팀으로 캡스톤디자인을 진행함에 따라, 팀워크 향상
